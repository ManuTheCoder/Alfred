# Alfred

Alfred is a multipurpose discord bot that lets you buy pets, roll dice, do coin flips, and more!

Features:

* Music
* Birthday Message Embeds!!!
* Coin flips
* Dice rolls
* Memes
* Custom Profile
* Random animal images
* Create Embeds
* Generate Images
* Create memes
* React to messages
* Pets
* Rock Paper Scissors
* Random images
* Slot Machine
* 8 Ball
* XP system
* Jobs
* Beg for money
* Song lyrics
* Search YouTube for songs
